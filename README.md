# Postgress guide for MacOS


## Requirements
> **Install Brew**
> You need to install `brew` with this command:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
